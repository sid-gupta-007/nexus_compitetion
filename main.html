About This Project

This website was created as part of NeXus 3.0, a frontend web design and development competition hosted by the IETE Students Forum.

The website is entirely front-end designed using HTML, CSS, and JavaScript without any pre-built templates.

Key Features

  • Fully responsive design
  • Created from scratch using HTML, CSS, and JavaScript
  • No pre-built templates used
  • Hosted on GitHub Pages
  • Code stored in a public GitHub repository

Submission Guidelines

  • Submit the link to the hosted site
  • Submit the link to the GitHub repository
  • Submit a zip file containing the website code
  • Ensure no commits are made post-submission
  • Provide a detailed README or a PDF explaining functionality
